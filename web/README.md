# 🚀 Veido SaaS - Discord Bot Dashboard

Plataforma tipo SaaS para administrar bots de Discord con dashboard web, sistema de niveles, módulos y API propia.

---

## 🌐 Demo

- Frontend: https://veido-dashboard.vercel.app
- Backend API: https://veido-saas.onrender.com

---

## ⚙️ Arquitectura

- 🎨 Frontend: React (Vercel)
- ⚙️ Backend: Node.js + Express (Render)
- 🤖 Bot: Discord.js
- 🗄️ Database: MongoDB

---

## 📦 Features

- 🔐 Login con Discord OAuth2
- 📊 Dashboard de servidores
- ⚙️ Activar/desactivar módulos (levels, welcome)
- 🧠 Sistema de XP y niveles
- 📡 API conectada en tiempo real
- 🤖 Bot de Discord funcional
